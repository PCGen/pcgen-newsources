PCGen 4E Data Set Creation: Data File Structure

In order to have pcgen load your files you must follow certain standards.

For our data set, you must keep them organized and easy to find.

Naming conventions are essential to good organization and grouping.

Today we will expand that to making homebrew sets, mainly 4th edition.

To start locate your pcgen folder (default windows is c:/program files/pcgen/pcgenxxxx/) Inside the pcgen folder is the version you are using, and then a dir structure as with our focus on these two:

  • data - where we hold the book information in the form of lst and pcc fils.
  • system - where we will have the gamemode inforamtion

The Data Folder

We will focus first on the data folder. Inside PCGen's data folder you will see the following subfolders:

alpha
This folder is where PCGen's 'alpha' data sets are found.
customsources
This folder is where data sets created by PCGen's LST Editor are stored.
d20ogl
This folder is where PCGens officially distributed data sets are found.
my_dataset
This folder is where 'homebrew' data sets are generally placed.
publisher_logos
This folder contains publisher logo artwork which is displayed in PCGen's 'Source Information Panel'

I suggest you create a new folder called 'homebrew' and inside your new homebrew folder create a folder called '4e'.

I like simple names but you can be as elaborate as you wish. I suggest that you use the underscore (_) as a substitute for the space in your folder name so if, for example, you use '4th_edition' as your folder name you won't have issues referencing it in the future.

Inside your newly created folder, henceforth called the 4e folder, we will create the required text files using the following naming convention: gamemode_lsttype.extension.

There are two extension that are used in data files. These are the ",pcc" and the ".lst" extensions.
- the .pcc is the control file that pcgen will use to identify all lst files to be used in a campaign.
- the .lst file is our basic file format for the data files that define the objects, e.g. classes, races, feats, etc. used within PCGen.

Examples of the files you will be creating are as follows:

  • 4e.pcc
  • 4e_classes.lst
  • 4e_races.lst
  • 4e_abilities.lst
  • 4e_feats.lst
  • 4e_equipment.lst
  • 4e_languages.lst
  • 4e_skills.lst
  • 4e_kits.lst
  • 4e_profs_armor.lst
  • 4e_profs_shields.lst
  • 4e_profs_weapons.lst

You will need at least one file of each kind.
